## Project Overview

Earshot is a Raspberry Pi 4B application that records conversations locally, diarizes speakers, and transcribes audio — entirely on-device. See `README.md` for a full description.

**Status:** Architecture and requirements phase — source code not yet written.

- `docs/adr/` — All architecture decisions. Treat these as authoritative on technology choices and design rationale.

- `docs/requirements/open-questions.md` — Unresolved decisions; flag these rather than assuming.
- `docs/development-workflow.md` — How to develop, test, and deploy.
