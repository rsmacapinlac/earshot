# Earshot — Requirements Index

Earshot is a Raspberry Pi application (Pi 4B or Pi Zero 2W) that records audio via a ReSpeaker 2-Mic Pi HAT and stores recordings locally. Recordings are offloaded physically via USB — no network connectivity is required for application functionality.
